# Production-Analysis-Dashboard-
## 📌 Project Overview

This project is an interactive **Production Working Hour Dashboard** designed to analyze workforce productivity, monitor working hours, and identify inefficiencies in production processes.

It converts raw production data into **clear visual insights** that help management make better operational decisions.

---

## 🎯 Objectives

- Track total working hours vs productive hours  
- Identify idle time and inefficiencies  
- Analyze employee and shift performance  
- Monitor daily, weekly, and monthly productivity trends  
- Support data-driven decision-making in production management  

---

## 📊 Key Features

✔ Working hour vs productivity comparison  
✔ Employee/shift performance tracking  
✔ KPI monitoring dashboard  
✔ Visual charts and performance breakdowns  
✔ Easy-to-understand business insights  

---

- Microsoft Excel
